恶名录：激进派背景包

“我做过可怕的事，那些事我永远不会对你、也不会对任何从这儿到黄金王座之间的人提起。我做了那些事，它们便成了我的枷锁，我永远无法挣脱。”

——容特·耶格（Junt Yaeger），在向审判官赫罗德（Herrod）的私人告解中如是说

以下章节详细介绍了《黑暗异端》角色可选的多种背景包。这些背景通过“填充”角色在加入审判庭之前的更多过往与经历，为侍僧增添了额外的细节层次，为角色提供了属于他们自己的“故事”，并附带故事线索，帮助玩家和游戏主持人构建其性格、信念与目标的基础，更遑论解释了他们为何效忠于一位激进派审判官，或为何自身便带有激进派倾向。本节中概述的所有背景包，在某种程度上都受到黑暗的玷污或触碰，从而将角色塑造成适合为神圣修会中的激进派分支服务的形象。取决于角色的职业，背景包可能表现为过去的一项黑暗发现、一个隐藏多年的可怕秘密，或是一件悄然犯下的邪恶行径。无论形式如何，这一事件都在角色的发展历程中标志着一个关键转折点。

除了为角色提供一段过往历史外，每个背景包还附带相应效果，以特定的技能、天赋或特性（有些相当独特）的形式呈现，代表角色因过往经历而发展出的能力。在许多情况下，这些背景包也包含相当严重的缺陷，通常会削弱某一能力，甚至给予失心或堕落点数。毕竟，第41个千年乃黑暗时代，一切知识皆有代价。

使用背景包 Using Background Packages

背景包会提供一系列属性调整、技能、天赋，有时还会赋予新的特性。背景包为角色带来的任何调整，不计为提升等级（Rank）所需的进阶（Advance），即购买背景包所花费的经验值不计入升级所需经验值，且其带给属性值的增减，也不计入通过正常手段进行的提升。

要选择一个背景包，侍僧必须满足其先决条件。部分背景包仅适用于特定职业，而另一些可适用于多种职业。许多背景包还要求侍僧必须来自特定的出身或世界。最后，所有背景包都需要花费经验值购买，且必须从创建角色时的起始经验值中支付。该价格涵盖整个背景包，玩家不能仅花费较少的经验值购买其中一部分，即，玩家要么全选，要么不选。同样，玩家可能会发现背景包授予的某些天赋，在常规情况下受属性限制而无法选取。侍僧无需满足背景包给予的天赋所列出的任何先决条件，毕竟这些奖励无疑是他们辛苦挣来的！

玩家只能在创建角色时获取背景包。玩家不能在角色职业生涯后期回头再选择一个背景包，也不能拥有超过一个背景包。是否启用背景包规则由游戏主持人决定，玩家在为自己的角色选择背景包之前，应始终征得其游戏主持人的许可。

示例 Example
艾玛创建了一个帝国世界文员角色，并希望选择一个背景包，以便让她的角色融入游戏主持人正在进行的激进派战役。她查阅了其职业可用的背景包，并选定持尔真名Thy Name I Keep。在获得游戏主持人许可后，她将背景包的效果应用到她的角色上。她将效果下列出的额外技能与天赋添加到角色卡上，并投掷 1d5+5 点堕落和 1d10 点失心，象征其知晓了某个恶魔真名的一部分。她从游戏主持人给予的起始经验中扣除400点，用以支付背景包的价格，然后继续花费余下的起始经验。

当然，玩家可以根据自己的喜好和想法，改变背景包中的“故事情节”。实际上，规则鼓励玩家这样做——只要效果和花费与所列内容相同即可。为求多样性，也建议一个团队中不要有两名玩家角色选用相同的背景包，除非团队的游戏主持人另有某种巧妙险恶的计划……
